## Covid Apis

This repository contains simple Covid web Apis

- [Documentation](https://arcane-chamber-95694.herokuapp.com/)

## Instalation

- `git clone https://github.com/faisalfani/covidApis.git`
- `cd covidApis`
- `npm install`
- `npm start`
- Visit http://localhost:3000
